PiTiTiCo - Etapa 1: Isolamento de dados por usuário

Correções:
1. Dados pessoais agora ficam separados por usuário/e-mail:
   - Bebês
   - Rotinas
   - Atividades
   - Rede de apoio
   - Alimentação
   - Agenda

2. Ao trocar usuário, o APP limpa a tela e carrega apenas os dados daquele login.

3. Frase alterada para:
   "Para o melhor bebê do mundo."

4. APP configurado para o servidor Render:
   https://pititico-servidor.onrender.com

Como testar:
1. Entrar com usuário A.
2. Cadastrar bebê, rotina, agenda etc.
3. Trocar para usuário B.
4. Confirmar que os dados do usuário A não aparecem.
5. Voltar para usuário A.
6. Confirmar que os dados dele voltam.
