PiTiTiCo v1.5 - Anúncios Admin Aleatórios

Alterações:
1. O anúncio do início do login usa os anúncios cadastrados no Admin.
2. O anúncio após login/cadastro também usa os anúncios cadastrados no Admin.
3. A cada reload, fechamento/abertura do browser ou APP, o anúncio é escolhido aleatoriamente.
4. Não usa mais trava de "visto hoje" para esses anúncios.
5. Mantém gratuito 90 dias e sem popup de atualização.
6. Badge visual permanece v1.0.

Substituir no Render/GitHub:
public/index.html
public/app.html

Pode manter:
server.js
admin.html
versao.json

Observação:
Para controlar os anúncios, use o Admin normalmente:
- título
- texto
- imagem
- link
- status ativo
